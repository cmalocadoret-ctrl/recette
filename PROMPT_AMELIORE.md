# Prompt amélioré — site 3D premium Ansamble

Crée un vrai mini-site web de présentation, pas un simple fichier HTML statique, pour présenter mes projets phares 2025/2026 chez Ansamble. Le rendu doit remplacer un PowerPoint et fonctionner comme une expérience narrative premium autour de la métaphore de la recette de cuisine.

## Stack technique attendue

Utilise un vrai projet front prêt à déployer sur Vercel : React + Vite ou Next.js. Le rendu doit intégrer Three.js / React Three Fiber pour les éléments 3D, GSAP ou Framer Motion pour les transitions, et du CSS avancé pour le design éditorial. Le livrable doit inclure au minimum `package.json`, `index.html`, `src/App.jsx`, `src/styles.css` et un README de déploiement.

## Direction artistique

Je veux un univers proche de la référence visuelle fournie : une grande carte héro premium avec fond photographique culinaire ou rendu 3D de légumes, barre supérieure noire arrondie, boutons orange terracotta, fond extérieur flouté, grande typographie serif, ambiance marque / formation premium. Couleurs : crème papier naturel, vert profond Ansamble, terracotta, brun chaud. Le résultat doit être mature, professionnel, institutionnel, jamais cartoon, jamais dashboard, jamais scolaire.

## Expérience attendue

Structure en scènes plein écran avec navigation :
1. Introduction immersive : carte héro façon site premium, titre "Ma recette logistique 2025/2026", sous-titre "Rendre les flux lisibles, les données fiables et les pratiques partageables".
2. Mise en place : 4 axes de l'année, révélés avec animations sobres.
3. Ingrédients : marmite 3D centrale, ingrédients cliquables, trajectoire fluide vers la marmite, vapeur, jauge de progression.
4. Cuisson : four 3D, animation chaude, étapes Observer / Cartographier / Structurer / Tester / Standardiser.
5. Dressage : assiette 3D, cloche qui se lève, bénéfices finaux.
6. Conclusion : phrase forte finale.

## Contenu projet à intégrer

Les 3 projets phares sont :
- Réorganisation logistique Grand Ouest / multi-sites.
- Digitalisation du portage de repas.
- Structuration de la donnée logistique — Ansamble 2030.

Le message central : mon année a consisté à transformer une logistique encore très terrain en une logistique plus structurée, plus fiable et plus pilotée par la donnée.

## UX obligatoire

- Notes orales masquées par défaut dans un panneau latéral.
- Navigation clavier : droite/espace pour avancer, gauche pour revenir, N pour notes, F plein écran, R rejouer.
- Responsive desktop prioritaire.
- Aucun élément ne doit se superposer.
- Pas de frise horizontale figée.
- Pas de texte long au centre des scènes.
- Le site doit donner l'impression d'un support de présentation haut de gamme.
